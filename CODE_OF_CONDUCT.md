# Code of Conduct
## Introduction
This is the Code of Conduct for the 
members of the Magimals Toxic group. 
## Strikes
The system revolves around 'strikes' - 
punishments for bad deeds. 
## Punishments
Fake insult - Warning. 
Insults - 1 strike. 
Threats - 2 strikes. 
Physical Harm - 3 strikes. 
## Firing
5 strikes = You're fired.
