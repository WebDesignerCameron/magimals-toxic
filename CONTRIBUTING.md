# Contributing
## Introduction
To contribute to Magimals Toxic, join the group. 
You can do that via WDC Ask.
You can also make a PR, if you
specifically want to join the
programming side of things. 
## WebDesignerCameron Ask
Send your request on 
[this website](https://webdesignercameron.github.io/WebDesignerCameronAsk),
and make sure to link your email
address so we can get in touch. 
## Forks
If you dont have git and gh on
terminal, execute
```bash
sudo apt update
sudo apt install git gh
git --version
gh --version
gh auth login
# Follow the prompts given
# Select GitHub.com
# Choose your protocol, https or ssh
# Choose to open in a web browser
# Input the code given
git auth status
```
Navigate to the
[Magimals Toxic Repository](https://github.com/WebDesignerCameron/magimals-toxic),
or execute
```bash
git clone https://github.com/WebDesignerCameron/magimals-toxic
cd magimals-toxic
```
Click the fork button, located near
the star, or execute
```bash
gh repo fork WebDesignerCameron/magimals-toxic
```
Make your changes,
```bash
git push -u origin YOUR_BRANCH
```
and make a pull request, or execute
```bash
gh pr create --title "Your PR Title" --body "Description of changes"
```
